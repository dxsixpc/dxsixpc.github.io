# CSS布局：弹性盒子flex布局

### 弹性盒子模型

```css
弹性盒子模型
	是css3新增加的非常重要的用于布局的模块
	在这个模块当中增加了一系列的布局属性
	主要是针对子元素进行布局,排列和布局
	在css2当中，存在标准模式下的盒子模型和IE下的怪异盒子模型。这两种方案表示的是一种盒子模型的渲染模式。
	而在css3当中，新增加了弹性盒子模型，弹性盒子模型是一种新增加的强大的、灵活的布局方案。
	弹性盒子模型是css3中新提出的一种布局方案。是一种为了应对针对不同屏幕宽度不同设备的一整套新的布局方案。主要是对一个容器中的子元素进行排列、对齐和分配空白空间的方案的调整。
	在之前，css3曾经推出过旧版本的弹性盒子模型。相对于新版本的弹性盒子模型而言，旧版本的内容与新版本还是有些出入。
	而且，从功能上来讲，旧版本的弹性盒子模型远远没有新版本的盒子模型强大，从兼容性来讲，二者在pc端ie9以下都存在着兼容性问题，但是在移动端，旧版本的弹性盒子模型兼容性则更好一点。但是对于我们来说，我们依然要将主要的精力放在新版本的弹性盒子模型的身上，因为旧版本的弹性盒子模型淘汰是必然，随着手机端的兼容性逐渐提升，旧版本必将被淘汰。
	另外，新版本具有更加强大的功能，也值得我们进行深度的学习。
	那么我们对于新旧两个版本的弹性盒子模型，我们只需要抱着对比的心态学习即可，掌握新版本，了解旧版本，这样即使有一天我们需要使用旧版本，也可以非常容易的学习旧版本的弹性盒子模型。
	
	辅助概念:
		主轴:
			我们以元素排在一行为例，当元素排列在一行的时候，主轴既表示元素排列的方向，横向排列则主轴即可以理解为一条横线，又因为我们元素默认是从左向右排列，那么我们可以说在默认的情况下，元素的主轴的起始位置是在左，而方向为右，终点也为右。
			对应X轴,表示从左到右的方向,默认方向
		侧轴:
			元素垂直的方向即为侧轴。默认上为起点，下为终点。
			对应Y轴,表示从上到下的方向,默认方向
		弹性容器:
			我们想要使用弹性盒子模型，就需要将容器转换为弹性容器，我们说一个包含于子元素的容器设置了display:flex，那么这个容器也就变成了弹性容器。
			一个容器一旦设置了display:flex或playdiso:inline-flex.那么就叫做弹性容器
		弹性子元素:
			当子元素的父元素变成了弹性容器，那么其中的所有的子元素也自然而然的变成了弹性子元素。
			一旦一个子元素的父元素变成了弹性容器,那么其中的所有子元素都变成了弹性子元素
			(关系仅限于一层父子关系,子元素的子元素不是弹性子元素)
	如何使用创建(使用)弹性盒子模型
		只需要给一个容器级别的标签设置display:flex或display:inline-flex

	一旦容器变成弹性盒子模型,那么子元素就变成了弹性子元素,没有脱离标准文档流,也不会产生3px的bug
	
	弹性容器属性(给弹性容器设置的属性,操作的还是弹性子元素)
		flex-direction:
			设置弹性容器当中子元素的排列方式
			值:
				row:子元素默认排列在一行
				row-reverse:子元素排列在一行,反方向排列.相当于改变了主轴的方向,变成从容器最右边开始
				column:子元素默认纵向排列在一列
				cilumn-reverse:子元素排列在一列,反方向排列,相当于改变了侧轴的方向,变成从容器最下方开始
		flex-wrap:
			设置弹性元素的总宽度超过父容器时 是否换行处理
			默认特点:
				子元素如果整体宽度超过了父容器,并且没有设置换行,那么将会挤在一行,并且所有的子元素都会按照相同的宽度进行缩放.
			值"
				nowrap:默认值,规定元素不发生换行
				wrap:规定子元素在必要时换行
				wrap-reverse:规定子元素在必要时换行,反方向换行.相当于改变了侧的方向,上下方向改变
		flex-flow:
			当前属性是flex-direction和flex-warp的简写
			可同时使用上面两个属性的值,组合使用
		align-items:
			设置弹性主元素在侧轴上的对齐方式
			值:
				flex-start:表示弹性子元素在侧轴起始位置开始
				flex-end:表示弹性子元素在侧轴结束位置开始
				center:在侧轴的中间
				baseline:通过基线进行调整
		align-content:
			修改flex-wrap的行为,类似于align-items,针对的是行之间的对齐方式.多行之间
			值:
				flex-start:表示弹性子元素在侧轴起始位置开始
				flex-end:表示弹性子元素在侧轴结束位置开始
				center:在侧轴的中间
				space-between:左右两端的两个对齐紧靠左右两边,中间自动分配剩余空间
				space-around:均匀分配空间,两端不用完全靠近边框
				上下两端的两个对齐紧靠上下两边,中间自动分配剩余客供件
		justify-content:
			设置弹性子元素在主轴上的对齐方式
			值:
				flex-start:表示弹性子元素在侧轴起始位置开始
				flex-end:表示弹性子元素在侧轴结束位置开始
				center:在侧轴的中间
				space-between:上下两端的两个对齐紧靠左右两边,中间自动分配剩余空间
				space-around:均匀分配空间,两端不用完全靠近边框
				
	弹性子元素属性(属性设置在子元素上)
		order:	
			更改子元素的排列顺序
			number
				:在每个子元素属性上加上order:1/2/3/4;调整元素的位置,不改变其他属性.
		flex-grow:
			设置弹性盒子子元素的扩展比例
			值:
				number
					通过设置数字,表示所需占剩余空间的占比.然后占满剩余空间
			剩余空间:在容器当中没有使用的剩余空间,横向的剩余空间.
			计算元素扩展之后的宽度,
				首先计算出剩余总宽度
				计算出每一份剩余空间的宽度
				根据每一份元素占据的剩余空间的份数(number值),乘以每一份剩余空间的宽度再加上元素本身的宽度,就是扩展之后的宽度
		flex-shrink
			设置弹性盒子子元素收缩比例
			值:
				number,同等宽度下收缩比例越大,则收缩的数值越大
				一旦flex-shink值小于1,缩减的宽度就不足以使子元素完全缩在盒子里,会溢出
			计算元素的收缩值:
				计算溢出值
				计算总权重
				收缩的宽度=溢出值*((flex-shrink*宽度)/总权重)
		flex-basis:
			用来设置或检索弹性格子伸缩基准值
			值:
				auto:默认值
				长度或者百分比
			flex-basis:是widht的替代品
				如果子元素设置了flex-basis或者width,那么在分配空间之前,就会和父容器预约这么大的空间
					然后剩下的才是剩余空间,然后父容器再把剩余空间分配给flex-basis
				如果同时设置了flex-basis和width,那么width会被覆盖
			特点:
				flex-basis会影响弹性子元素(flex-items)的width属性,如果二者同时存在,flex-basis会覆盖width
				如果开启了wrap模式,那么检测弹性子元素是否满足换行需求时检测的是flex-basis的总值
				当flex-basis设置为auto时,将不会对width产生影响
				当flex-basis设置为百分比,参考的是弹性容器的宽度
		align-self:
			会覆盖设置在容器上的align-item属性
			具体操作单个的弹性子元素
		
		flex:
			是flex-grow,flex-shrink,flex-basis三个属性的缩写
			flex属性代表的三个属性的默认值和三个属性本来的默认值存在差异,mdn的解释为让flex更好用
			flex
			flex:设置弹性盒子的子元素如何分配空间。

			flex 属性是 flex-grow、flex-shrink 和 flex-basis 属性的简写属性。
				简介：
				flex是flex-grow,flex-shrink和flex-basis的缩写，flex属性值可以只指定一个属性的值，而另外的属性值采
				用各自在flex属性中的的初始值，但是有一点要注意的是：flex属性中flex-grow和flex-basis的初始值和它们原始
				的默认值不同，至于为什么不同，mdn中有明确的说过这样的设计是为了让「flex」缩写在最常见的情景下比较好用。

			flex中对应各属性的初始值：
				flex-grow
				  flex-grow用于设置各item项按对应比例划分剩余空间，若flex中没有指定flex-grow,则相当于设置了
				flex-grow:1

				flex-shrink
				  flex-shrink用于设置item的总和超出容器空间时，各item的收缩比例，若flex中没有指定flex-shrink,
				则等同于设置了flex-shrink:1

				flex-basis
				  flex-basis用于设置各item项的伸缩基准值，可以取值为长度或百分比，如果flex中省略了该属性，则相当
				于设置了flex-basis:0.

			flex的不同取值
				  flex的值的完整写法是[<flex-grow> <flex-shrink> <flex-basis>],不过它的取值还有可能是单个数字
				或者单个百分比等，不同种类的取值所表示的意思是大有不同的。

				flex值为none
				  当flex为none时,等同于flex: 0 0 auto;

				flex值为auto
				  当flex为auto时，等同于flex: 1 1 auto;

				flex值为一个非负数字
				  当flex取值为一个数字，则该数字是设置的flex-grow值，其它两个属性用初始值，如flex:1时，
				等同于flex: 1 1 0%

				flex值为两个非负数字
				  当flex取值为2个数字时，相当于设置的flex-grow和flex-shrink值，flex-basis取值为初始值，
				如flex:1 0时，等同于flex: 1 0 0%

				flex值为一个数字和一个长度或百分比时
				  当flex取值为1个数字和1个长度或百分比时，设置的是flex-grow和flex-basis的值，flex-shrink值
				时初始值，如flex:1 20%,等同于flex: 1 1 20%
```
