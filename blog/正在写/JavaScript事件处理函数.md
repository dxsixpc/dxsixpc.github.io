## 事件处理函数

```
主要用于和用户产生一定的交互，提高页面或程序的交互性。

    事件和函数之间的关系
       当触发事件的时候，执行函数－－－－－－－按下制冷按钮，空调吹冷风

    事件－－－－调用－－－－函数

事件和函数的关系，
	用户通过触发事件，执行对应的函数
	事件，执行，函数
	函数，事件处理函数
	
事件源：
	触发事件的源头
事件类型：
	什么行为:
		点击，双击。。。。。
事件处理函数，触发事件源的对应的时间类型时，直接被执行的函数
	当前统一语法：
		事件源.事件类型 = 事件处理函数
```
